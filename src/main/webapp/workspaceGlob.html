Specify the files to include in the archive of the build's workspace, using wildcards like 'module/dist/**/*.zip'. See the @includes of Ant fileset for the exact format. The base directory is the workspace.

Option Specific build number was added to be able to select an specific workspace to restore.

  • Build Number: Is the build number of the job that store the workkspaces to be restored. It will only be used when Specific build number is selected.
